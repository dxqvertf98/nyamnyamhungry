memo

- 목적: 학교 축제를 위한 안내 웹사이트 제작


- 구성: 부스의 위치를 표시한 지도, 부스 찾기, 행사 안내 (세부사항은 분야별로 밑에 설명함)

    - 부스 지도
        - 우리 학교 지도를 띄움.
        - 최하단에 작은 네모 버튼 안에 각각 1, 2, 3, 4, 5 적힘 -> 각 층별 지도로 이동하는 버튼
        - 학교 지도 안에 부스 이름 띄움
            - 부스 이름 클릭 시 해당 부스에 대한 세부 설명 창 혹은 팝업으로 이동
            - 세부 설명 창
                - 최상단에 제목으로 부스 이름 적어둠
                - 밑에 회색 글씨로 동아리 이름 작게 적음
                - 그 밑에 키워드 추가 (ex.음식, 게임, 사진...) -> 밑에 부스 찾기에서 키워드별 검색 가능
                - 본문에는 부스 위치(ex. 4층 2-3반), 활동 내용, 설명이 들어감
                - 팝업의 형태일 시 오른쪽 최상단에 X 표시를 누르면 다시 축제 부스 지도로 돌아감
                
    - 부스 찾기
        - 검색창에 키워드(ex. 과학, 요리, 방탈출...), 동아리명, 부스명에 들어가는 단어로 검색 가능
        - 검색 시 해당하는 것으로 예상되는 부스의 부스명,동아리명,키워드를 네모 안에 묶고 제일 및에 더 알아보기 버튼
        - 클릭 시 축제 부스 지도에서 사용했던 세부 설명 창으로 이동
        
    - 학교 행사 안내
        - 시간대로 표를 만들어서 장소와 시간 설명 (ex. 밴드부 공연 | 강당 | 14:00 ~ 15: 00 )

- 작동 과정
    - 사이트 진입 시 처음에는 로비 화면으로 이동함
    - 로비 화면에는 "부스 지도", "부스 찾기", "학교 행사 안내" 라는 버튼 3가지가 있음
    - 클릭 시 각각 구성의 부스 지도, 부스 찾기, 학교 행사 안내로 이동함
